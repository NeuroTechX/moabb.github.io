Analysis¶

moabb.analysis.analyze(results, out_path, name='analysis', plot=False)[source]¶

Analyze results.

Given a results dataframe, generates a folder with results and a dataframe of the exact data used to generate those results, aswell as introspection to return information on the computer

Parameters

  • out_path (location to store analysis folder)

  • results (Dataframe generated from Results object)

  • path (string/None)

  • plot (whether to plot results)

  • Either path or results is necessary
